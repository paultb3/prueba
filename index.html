Conceptos para programar como un crack

Variables

Son contenedores que almacenan datos que pueden cambiar durante la ejecución del programa.

Condicionales

Permiten que el programa tome decisiones basadas en ciertas condiciones (if, else, switch).

Bucles

Se usan para repetir bloques de código mientras se cumpla una condición (for, while).

Funciones

Bloques de código reutilizables que realizan una tarea específica cuando son llamados.

Arrays

Son estructuras que almacenan múltiples valores en una sola variable.

Objetos

Estructuras que permiten almacenar datos y funciones relacionados en una sola entidad.

Eventos

Acciones que ocurren en la página (como clics o teclas presionadas) y pueden ser detectadas y respondidas por el código.

Promesas

Objetos que representan la eventual finalización (o fallo) de una operación asíncrona.
